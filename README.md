# Admin for portfolio website
## Requirements
- node version `v17.5.0`

## Setup Instructions
- clone the repository
- `cd portfolio-admin`
-  `yarn install`
-  `yarn start` to run project

or just use docker:
```bash
$ docker-compose up -d --build
```
Navigate to http://localhost:3000 in your browser
